## 4. Risk-Based Prioritization

1. **Critical path** — Full query pipeline (Auth → Cache → Parallel BM25+Vector → RRF → Rerank → Response); authentication flows; MCP protocol
2. **Integration boundaries** — Cache invalidation on reindex; language filter applied to both BM25 and Qdrant simultaneously; repo-scoped filter; auth middleware → query handler
3. **NFR thresholds** — Performance (latency p95, throughput QPS) under real system load; security (auth bypass, input injection)
4. **UI/A11y** — Web UI accessibility (WCAG 2.1 AA); branch selector; search results
5. **Compatibility** — Python 3.11/3.12 dual-version; browser compatibility (Chrome, Firefox)
