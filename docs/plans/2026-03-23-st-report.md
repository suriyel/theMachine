## 7. Recommendations

1. **Docker images (NFR-012)**: Add `Dockerfile` for `query-api`, `mcp-server`, and `index-worker` services and wire them into the build/CI pipeline. This is the only deferred "Should" requirement.

2. **Real infrastructure tests in CI**: The 19 skipped tests (`pytest.mark.real`) require live ES/Qdrant/Redis/Postgres. Add a separate CI job that provisions services and runs the full suite including real-marked tests.

3. **Reranker monitoring**: Add a Prometheus counter for reranker fallback events (D3 fix logs a warning; a metric would enable alerting on model degradation).

4. **Query size limit**: Queries up to 10,000+ characters are accepted (Charter 9). Consider adding a max-length Pydantic validator to prevent excessive tokenization cost in the embedding/reranker models.

5. **Python 3.11 matrix test**: Add a GitHub Actions matrix entry for Python 3.11 to continuously validate backward compatibility.
